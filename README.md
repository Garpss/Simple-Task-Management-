# Simple Task Board

## About this app

This app is a simple task Management system where
You can:

- Add a task
- Edit or delete a task
- Drag and drop tasks between columns
- Set assignee, deadline, priority, points, and tags
- See deadline status (Overdue, Due today, Due soon)
- Search and filter tasks

## Tech used

- React
- TypeScript
- Vite
- LocalStorage

## How to run

```bash
npm install
npm run dev
```
